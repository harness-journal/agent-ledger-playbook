# Solo Creator Example

## Example Goal

혼자 AI를 써서 글쓰기, 리서치, 정리를 하는 사람이 최소 기록 시스템을 붙이는 예시.

## Minimum Setup

- `templates/work-log.md` 복사
- 작업이 끝날 때마다 한 블록씩 추가

## Example Entry

## 2026-04-02 15:20
- task: Simon Willison 4월 글 Threads 초안
- actor: inabe
- model: gpt-5
- prompt: 사이먼 4월 글을 비개발자도 이해하게 설명해줘
- response: 운영, 추적, 권한 중심으로 쉽게 정리
- tools: web_search
- next_action: 발행용 최종 다듬기
- artifact: threads_draft_v1.md
